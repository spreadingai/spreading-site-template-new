<Title>如何处理文件共享 SDK 所有接口均异常的问题？</Title>


---

## 问题描述

集成文件共享 SDK，初始化后，所有接口均调用失败。

## 问题原因

文件共享 SDK 所有接口均调用失败一般可能是初始化传入的参数异常。

## 解决方案

针对接口调用失败的问题，ZEGO 提供了以下排查方法：

1. 检查初始化传入的参数的格式和合法性。
2. 根据 SDK 返回的错误码进行排查。
