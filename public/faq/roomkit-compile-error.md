如果调用 SDK 接口时，控制台报了类似上图的错误，可能是因为对 SDK 进行了编译打包导致的错误。

建议参考 [集成](https://doc-zh.zego.im/article/8111) 文档通过 script 的方式引入 SDK，不对 sdk 进行编译打包。
