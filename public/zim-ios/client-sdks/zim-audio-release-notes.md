# ZIM Audio 发布日志

- - -

## 1.0.0 版本

**发布日期：2024-04-17**

ZIM Audio SDK 首次发布，支持完整的语音处理功能，包含语音采集、播放、噪声抑制（ANS）、自动增益控制（AGC）等，开发可以轻松实现高清语音消息的收发，无需关注音频处理的底层实现。若需使用，请同时搭配 ZIM SDK 接入。
