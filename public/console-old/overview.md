# 概览

- - -

欢迎使用 [ZEGO 控制台](https://console.zego.im)，控制台为您提供实名认证、项目管理、用户与权限、用量统计等功能的配置和管理操作。

<Warning title="注意">



**2021-11-16** 之后注册 [ZEGO 控制台](https://console.zego.im) 的用户，请参考控制台的 [控制台概览](/console/overview) 及相关文档。


</Warning>
