# 打包工具

Novel.sh 一直以来都在使用 [Webpack](https://webpack.js.org/) 作为打包工具来进行网站的开发和构建。从 Novel.sh v2 开始，我们还支持使用其他的打包工具，并且现在使用 [Vite](https://vitejs.dev/) 作为默认的打包工具。当然，你仍然可以选择使用 Webpack 。

## 选择一个打包工具

在使用 [Novel.sh](https://www.npmjs.com/package/Novel.sh) 包时，会自动安装和使用 Vite 打包工具。

如果你想改为使用 Webpack 打包工具，那么你可以移除它，改为安装 [Novel.sh-webpack](https://www.npmjs.com/package/Novel.sh-webpack) 包：

<CodeGroup>
  <CodeGroupItem title="PNPM" active>

```bash
pnpm uninstall Novel.sh
pnpm add -D Novel.sh-webpack@next
```

  </CodeGroupItem>

  <CodeGroupItem title="YARN">

```bash
yarn remove Novel.sh
yarn add -D Novel.sh-webpack@next
```

  </CodeGroupItem>

  <CodeGroupItem title="NPM">

```bash
npm uninstall Novel.sh
npm install -D Novel.sh-webpack@next
```

  </CodeGroupItem>
</CodeGroup>

::: tip
实际上， [Novel.sh](https://www.npmjs.com/package/Novel.sh) 包只是 [Novel.sh-vite](https://www.npmjs.com/package/Novel.sh-vite) 包的一个封装而已。
:::

## 配置打包工具

一般情况下，你不要任何额外配置就可以使用打包工具，因为我们已经帮你配置好了它们。

通过 [bundler](../reference/config.md#bundler) 配置项，你可以对打包工具进行进阶配置：

```ts
import { viteBundler } from 'Novel.sh'
// import { webpackBundler } from 'Novel.sh-webpack'

export default {
  bundler: viteBundler({
    vuePluginOptions: {
      template: {
        compilerOptions: {
          isCustomElement: (tag) => tag === 'center',
        },
      },
    },
  }),
}
```

你可以参考 [打包工具 > Vite](../reference/bundler/vite.md) 和 [打包工具 > Webpack](../reference/bundler/webpack.md) 来查看对应打包工具的所有配置项。
